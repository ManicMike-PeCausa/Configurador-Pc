Configurador PC

Sistema de Configuración de PCs - Proyecto Integrado FP

Aplicación de escritorio desarrollada en Java para la configuración y compra de componentes de PC

Ver Proyecto GitHub

Sobre el Proyecto

Descripción

Configurador PC es una aplicación de escritorio desarrollada como proyecto integrado para el ciclo de FP. Permite a los usuarios configurar PCs personalizados seleccionando componentes compatibles.

El sistema incluye gestión de usuarios, validación de compatibilidad entre componentes y un panel de administración completo.

Objetivos

  • Aplicar conocimientos de programación en Java
  • Implementar arquitectura MVC
  • Trabajar con bases de datos MySQL
  • Desarrollar interfaces gráficas con Swing
  • Usar control de versiones con Git
  • Generar documentación automática con Javadoc
  • Crear facturas en formato XML

Funcionalidades Principales

👤 Gestión de Usuarios

Sistema de login con roles diferenciados (cliente/administrador)

🛒 Carrito de Compra

Selección de componentes con cálculo automático del total

✅ Validación

Verificación automática de compatibilidad entre componentes

📊 Panel Admin

Gestión completa de productos, precios y stock

💰 Sistema de Créditos

Los usuarios tienen créditos para realizar compras

📄 Facturas XML

Generación automática de facturas en formato XML

Arquitectura MVC

src/
├── modelo/
│   ├── Usuario.java
│   ├── Componente.java
│   └── ComponenteDAO.java
├── vista/
│   ├── LoginFrame.java
│   └── MainFrame.java
└── controlador/
    └── MainController.java

El proyecto sigue el patrón Modelo-Vista-Controlador para separar la lógica de negocio de la interfaz.
